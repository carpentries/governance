# About
The following file includes motions passed during governance meetings from 2022 and onward. To see motions passed in previous meetings, navigate to the [minutes](https://github.com/carpentries/governance/tree/main/minutes) from the previous years. This file is updated quarterly based on the minutes of the quarterly Board meetings.
